# GitHub Pages — Privacy Policy

Publish the privacy policy for **Proverbs 31: Daily Organizer Pro**:

1. Push this repository to GitHub.
2. Open **Settings → Pages**.
3. Source: **Deploy from a branch**
4. Branch: `main` → folder **`/docs`**
5. Save. Your live URL will be:

   `https://christianappsfamily.github.io/Proverbs31DailyOrganizerPro/privacy-policy/`

Update `expo/constants/urls.ts` → `PRIVACY_POLICY_URL` if the repo name or org changes.

The policy HTML lives at `docs/privacy-policy/index.html`.
