# GitHub Pages — Privacy Policy

Publish the privacy policy for **Proverbs 31 Organizer**:

1. Push this repository to GitHub.
2. Open **Settings → Pages**.
3. Source: **Deploy from a branch**
4. Branch: `main` → folder **`/docs`**
5. Save. Your URL will be:

   `https://<your-github-username>.github.io/<repo-name>/privacy-policy/`

Update `expo/constants/urls.ts` → `PRIVACY_POLICY_URL` to match your live URL.

The policy HTML lives at `docs/privacy-policy/index.html`.
